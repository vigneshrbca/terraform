# Terraform AWS Infrastructure 🚀

This project provisions a basic AWS infrastructure using Terraform.

## 🧱 Resources Created
- VPC
- Public Subnet
- Internet Gateway
- Route Table
- Security Group (SSH access)
- EC2 Instance

## ⚙️ Prerequisites
- AWS CLI configured
- Terraform installed

## 🚀 Usage

```bash
terraform init
terraform plan
terraform apply
